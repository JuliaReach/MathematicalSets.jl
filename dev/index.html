MathematicalSets.jl

MathematicalSets is a Julia package for mathematical sets interfaces.

Features

  • An abstract type for mathematical sets.
  • Generic and flexible functions to operate with sets.

Usage

The following packages implement the MathematicalSets.AbstractSet interface:

LibraryDescription
LazySetsA Julia package for calculus with convex sets
PolyhedraPolyhedral Computation Interface
SemialgebricSetsExtension of MultivariatePolynomials to semialgebraic sets
SetProgSet Programming with JuMP
